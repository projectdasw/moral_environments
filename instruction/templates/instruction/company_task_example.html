Contoh Halaman Company Task

Berikut adalah contoh untuk sebuah ronde di dalam Company Task dimana Anda harus memilih salah satu dari pilihan investasi untuk perusahaan yang Anda kelola.

Situasi: Perusahaan yang Anda berfokus pada usah perkebunan sawit dengan nilai kapitalisasi perusahaan sebesar 10.000 token dan akan melakukan ekspansi usaha dengan alokasi sebesar 1.000 token. Anda harus memilih prospek investasi untuk perusahaan yang Anda kelola, dan di satu sisi, Anda dapat berkontribusi untuk upaya minimalisir dampak negatif perkebunan sawit terhadap lingkungan yang mungkin dapat terjadi.

Prospek A: Investasi Perusahaan menghasilkan potensi return sebagai berikut:

  • Return sebesar 200% dengan probabilitas 50% atau
  • Return sebesar 0% dengan probabilitas 50%

Prospek B: Investasi Perusahaan menghasilkan potensi return sebagai berikut:

  • Return sebesar 150% dengan probabilitas 70% atau
  • Return sebesar 0% dengan probabilitas 20%

Potensi Kerusakan Lingkungan: Ada potensi kerusakan lingkungan sebesar 20 kali nilai investasi dengan probabilitas sebesar 5%. Jika kerusakan terjadi, maka Anda tidak mendapatkan return atas investasi yang dilakukan perusahaan.

Kontribusi: Berapa persen kontribusi yang perusahaan Anda akan berikan dari nilai return yang diperoleh jika tidak terjadi kerusakan lingkungan (tulis angka 0-100)
